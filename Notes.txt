Css Styling: 
3 ways:
1.inline   -> <h1 style="color:red"</h1> 
2.internal -> <style>
                    h1{
                        color: blue;
                        font-size: 30px;
                        font-family: arial-black;
                        background-color: beige;
                    }
            </style>
3.external
----
NOTE: inline style OVERRIDES internal or external styles
inline is dominant style
For example, if you want to override the color property,
use <h1 style="color:red"</h1>
Final styles :  color: red; 
                font-size: 30px;
                font-family: arial-black;
                background-color: beige;

class Attribute is used for applying the same styling for different elements
class attribute is the most common way to apply css styles
Use class when adding css style
. = class
----------------
id Attribute is used for applying UNIQUE styling for different elements
Becasue id MUST be unique
# = id

same style -> class
